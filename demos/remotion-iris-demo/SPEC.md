# Spec — Iris Email Operations Film

## Acceptance criteria

- 1920×1080, 30fps, 59.52 seconds, exact existing narration.
- Original seven-part Iris story: triage, property answer, search, seller path, send policy, memory/follow-up, close.
- Real product captures remain legible and integrated into authored UI.
- Every animated value derives from Remotion frame APIs. No CSS animation.
- Camera depth, type choreography, transitions, and object motion exceed existing flat scene fades.
- H.264 MP4 renders with AAC audio; ffprobe and contact-sheet verification pass.

## Anti-patterns

- No imitation of Shipper branding, copy, purple orb, exact sequence, or logo treatment.
- No ungrounded product capability claims.
- No tiny screenshot dump; each product capture gets a focused camera move and explanatory state.

## Scope

Complete standalone Remotion project. Hyperframes source stays untouched.

## Reference

User-supplied X-video frames and existing `demos/hyperframes-email-agent` assets/narration.
